# 声语信使 - 电子系统课程设计

基于 **4FSK 音频调制** 的数字短信息通信系统，通过声波实现两台 STM32 终端之间的数据传输。

## 项目信息

| 项目 | 详情 |
|------|------|
| 课程 | 电子系统课程设计 |
| 主控 | STM32F103C8T6 |
| 调制方式 | 4FSK（1500/1600/1700/1800 Hz） |
| 通信距离 | ≥ 0.5 米 |
| 码元时长 | 100 ms |

## 硬件架构

- **显示**：0.96" I2C OLED
- **输入**：4x4 矩阵键盘
- **发射**：PWM → MOSFET 低边驱动 → 扬声器
- **接收**：驻极体话筒 → MCP6002 运放放大 → ADC → Goertzel 检测
- **电源**：P-MOS + N-MOS 一键软开关，关机待机 < 1 mA
